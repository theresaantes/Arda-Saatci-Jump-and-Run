#README

Ein kleines Jump-and-Rung Spiel zur 600km-in96h-Challange von Arda Saatci. 
Arda läuft durch die Wüste und Stadt, sammelt Energie durch Gummibärchen-Würmer, weicht Hindernisssen aus und versucht rechtzeitig ins Ziel zu kommen. Am Ende warten seine Mama und ein Eiswagen auf ihn.
